# @gomessaging/nats

NATS transport implementation for the [gomessaging](https://github.com/sparetimecoders/messaging) specification (Node.js).

## Installation

```bash
npm install @gomessaging/nats
```

## Usage

See the [specification repo](https://github.com/sparetimecoders/messaging) for documentation and examples.

## Development

```bash
docker compose up -d
npm install
npm test
```
